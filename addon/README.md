# BetterPlayer

Аддон для визуальной кастомизации полноэкранного музыкального плеера. Расширяет стандартные возможности, добавляя гибкую настройку фона, новый дизайн панели управления и плавные анимации переходов.

## Основные возможности

### Внешний вид и фон
Все изменения фона сопровождаются **плавными анимациями**, что делает переключение треков более приятным.
- **Обложка как фон** — автоматическая установка обложки играющего трека на задний план.
- **Пользовательский фон** — возможность установить любое стороннее изображение.
  > ⚠️ **Важно:** Из-за ограничений Яндекс Музыки аддон поддерживает прямые ссылки на картинки только с проверенных хостингов. Вы можете загрузить свой фон на [ImgBB](https://imgbb.com/), [Postimages](https://postimages.org/) или [Catbox.moe](https://catbox.moe/), а затем вставить прямую ссылку в настройки.
- **Настройка размытия (Blur)** — точная регулировка размытия фона для улучшения читаемости текста.
- **Регулировка яркости** — ручная настройка яркости фона.
- **Нормализация яркости** — умный алгоритм автоматического затемнения светлых обложек до комфортного для глаз уровня.

### Интерфейс управления
- **Новый вид плеера** — переработанная, более эргономичная панель управления.
- **Стилизация элементов** — вы можете выбрать один из трех стилей подложки для панели управления:
